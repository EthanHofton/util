Detailed Description

client class. uses RIAA. when the client is created, it connects to the server and starts the listen thread. when it is deleted it disconnects and distroies the socket. However, the reconnect and disconnect will allow to do this early (the methods have been set up so that if disconnect/reconnect is called twice, it is ignored)
