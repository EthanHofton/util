Example

See also
event for the beginning part of the example

implamenting a simple event disptacher using the example events for the example section in event can be done in the following way:

bool handelType(typeEvent& t_event) { std::cout << "type event occured: " << t_event.toString() << std::endl; return false; }

bool handelClick(clickEvent& t_event) { std::cout << "click event occured: " << t_event.toString() << std::endl; return true; }

template<typename eventType> void onEvent(util::event<eventType>& t_event) { util::event_dispatcher<eventTypes> eventDispatcher(t_event);

eventDispatcher.dispatch<typeEvent>(&handelType); eventDispatcher.dispatch<clickEvent>(&handelClick); }

int main(int argc, char** argv) { typeEvent e('c'); clickEvent e2(123, 345);

onEvent(e); onEvent(e2);

return EXIT_SUCCESS; }

the onEvent method calls an event disptacher that will call the appropriate handler for each event. a main method will call the onEvent method with the generated event impls
